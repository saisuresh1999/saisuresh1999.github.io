# saisuresh1999.github.io

Personal academic website of **Sai Suresh Macharla Vasu**, built with the [Minimal Light](https://github.com/yaoyao-liu/minimal-light) Jekyll theme.

Live at: [https://saisuresh1999.github.io](https://saisuresh1999.github.io)

## About

This site serves as my academic homepage. It includes:
- About Me & Research Interests
- News & Updates
- Publications
- Experience, Education & Awards
- CV (PDF)

## How to Deploy (GitHub Pages)

This site is automatically deployed via **GitHub Pages**. Just push to the `main` branch and it goes live within a minute or two.

## Local Development (Optional)

If you want to preview locally:

```bash
# Install Ruby and Bundler first, then:
bundle install
bundle exec jekyll serve
# Open http://localhost:4000 in your browser
```

## Customization

| File | Purpose |
|------|---------|
| `_config.yml` | Name, position, email, links |
| `index.md` | About Me, Research Interests, News |
| `_data/publications.yml` | Publications list |
| `_includes/services.md` | Experience, Education, Awards |
| `assets/img/avatar.png` | Profile photo |
| `assets/files/curriculum_vitae.pdf` | CV PDF |
